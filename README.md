# Insurance Claim Analysis Dashboard

## Overview
This project presents an interactive Insurance Claim Analysis Dashboard built using Power BI to analyze insurance policies, claim settlements, customer demographics, and premium trends.

The dashboard helps in understanding claim distribution patterns, policy performance, and customer segmentation through visual analytics and KPI monitoring.

---

## Objectives
- Analyze insurance premium and claim settlement trends
- Monitor active and inactive insurance policies
- Identify high-claim customer segments
- Evaluate demographic patterns across policyholders
- Build interactive dashboards for insurance performance analysis

---

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization

---

## Dashboard Features

### Claim Analysis
- Total claim amount and settlement tracking
- Claim status distribution analysis
- Claim trend monitoring across insurance categories

### Customer Demographics
- Age-group analysis of policyholders
- Gender-based policy distribution
- Customer segmentation insights

### Policy Performance
- Active vs inactive policy monitoring
- Insurance category comparison
- Premium amount analysis by policy type

### KPI Monitoring
- Total premium collected
- Total claims settled
- Policy count and claim ratios
- Interactive filtering and drill-down analysis

---

## Key Insights
- Certain age groups showed higher claim frequencies compared to others
- Health and vehicle insurance categories contributed significantly to total claims
- A noticeable gap existed between premium collection and settlement amounts in specific policy segments
- Active policy distribution varied significantly across insurance categories
